# learning-cards
API description can be found here: https://github.com/KamilKrol5/learning-cards/blob/master/backend/API_description.md
### Technologies
* Python 3.7 and Django 2.2.8,
* React with Redux,
* Bootstrap 4.4.1,
* JWT authentication.
